# Smart Auth Dashboard

Simple login-based dashboard web application with API data (news, weather, crypto), interactive map, and dark/light mode using HTML, CSS, and JavaScript.

## Features

- Login and Signup validation  
- Password hashing (SHA-256)  
- Protected dashboard  
- Welcome message with username  
- News feed API integration  
- Crypto price API (Bitcoin, Ethereum)  
- Weather API integration  
- AI search assistant  
- Interactive map with user location  
- Dark / Light mode toggle  
- Toast notifications  
- Logout session handling  

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- LocalStorage  
- Public APIs

## How to Run

1. Download or clone the repository  
2. Open `index.html` in browser  
3. Register a new account  
4. Login to access dashboard  

## API Used

- News API / Hacker News API  
- CoinGecko Crypto API  
- Open-Meteo Weather API  
- DuckDuckGo Search API  
- OpenStreetMap (Leaflet Map)
